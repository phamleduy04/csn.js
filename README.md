# csn.js
A package to get music from chiasenhac.vn
## Installation
To install csn.js, use npm:
```
npm install csn.js
```
or yarn:
```
yarn add csn.js
```

## Documentation
The documentation of this package can be found [here](https://csnjs.nezukobot.vn/)
